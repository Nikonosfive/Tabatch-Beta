Tabatch Version 7.XX.x（Tabatch 7）

Version 7.42.3（2025.4.2）

・iPadでの動作改善のために画面表示切り替えの基準となる比率を0.8から0.67に変更

Version 7.42.2（2025.3.27）

・Apple製デバイス向けのアイコンを改善

・リリースノート用ページを作成（これに伴いサービスワーカーも修正）

Version 7.42.1（2025.2.27）

・10種類のビジュアルエフェクト機能を追加

・ドロップダウンメニューを整理

　1.表示調整機能を「Advanced Font Settings」のサマリー内にまとめた

　2.ソースコードを簡素化した

・ボタンや操作可能な要素にAlt（title属性）を付与した

・フルスクリーンモードの切り替えボタンの動作を改善

・コードのミスを修正

・時計の文字サイズを調整

Version 7.40.0（2025.1.25）

・適切にソースをキャッシュし、オフラインでも完全な機能が使用できるようサービスワーカーの動作を改善

・タッチパネルで操作する際に、左右フリックで機能を切り替えられるように改善

Version 7.30.0（2025.1.20）

・アイコンのデザインを一新、統一

・フォントウェイト調整機能を追加（フォント設定の「bold」を廃止）

・クリッカブルな要素にマウスポインターを当てた際に以下のいずれかの動作をするように改善

　1.マウスカーソルがポインターもしくは ][ になる

　2.opacityを低下させホバーを明らかにする

　3.scaleを変更してホバー時にわずかに拡大する

・ドロップダウンメニューのデザインを改善

　1.一部の囲い枠の太さを変更

　2.フルスクリーンモードにアイコンを設定

Version 7.20.0（2025.1.10）

・フォントサイズ調整機能を追加

・ラップボタンのデザインを改良（7.00.1以降崩れていた表示を改善）

・コードのミスを修正

Version 7.10.0（2025.1.8）

・フォントにVerdana / Google Sans Jersey 10 Charted / Robotoを追加

・フォントをGoogle Fontsから拾ってくる形式に変更（7.10.1より）

・PC表示で、横幅が小さい際にツールバーの機能名を非表示するように変更

・一部機能の表示を改善

Version 7.00.1（2025.1.7）

・スマホ表示とPC表示切り替えの基準となる画面比をやや横長に変更

・三点リーダの表示を修正

・Apple製デバイスでの表示を改善

Version 7.00.0（2024.6.17）

・Cookieに本格対応し、基本的な設定項目を全て記憶するように改善

・ドメイン移転「watch.seieisai37.com」→「watch.tadr-tech.info」

・本格的なラップ機能を追加

・キーコントロール機能割当を改善

・初期状態での表示を改善

・Apple製デバイスでのフォントを改善
